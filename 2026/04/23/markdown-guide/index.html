上手必看：Markdown的使用教程

关于markdown

Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写，Markdown被大量使用，如Github、Wikipedia、简书等。如github的readme.md(项目介绍文件）就是由Markdown进行编写的

包括飞书也是可以支持markdown进行笔记和编辑的，他可以让你在短短的10几个符号就完成对一篇笔记的编写。

其他的Markdwon编辑软件
除了飞书，还要许多不同的Markdown的编辑软件，提供选择以下列举几个比较常用的Markdown软件

Markdown编辑软件

Notion

相比于飞书，Notion更加专注于作为一个笔记软件，并且支持多端同步查看，多人团队协作，丰富的API接入口允许你从github，Gmail等平台快速导入模板与数据，但是和飞书一样有个缺点就是需要将文件上传到云端才可以正常存储，但是在更新以后可以支持离线查看了，但是依然没有解决直接查看本地文件的问题
传送门：
https://www.notion.com/
obsidian

这是一个适合本地编辑的Markdown软件，不同于飞书，它运行我们在本地直接创建仓库，直接打开本地文件进行编辑，但是缺点是它无法使用颜色字体和没有实时渲染
传送门：
https://obsidian.md/
Visual Studio Code
哈哈没想到吧，vs code不仅仅是一个编程软件它也可以进行Markdown的编写，我们只需要安装插件Markdown Preview这个插件就可以进行Markdown的预览了
